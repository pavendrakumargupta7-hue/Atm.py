# ATM System (Atm.py)

## Project Description
- This project is a Python-based ATM System simulation.
- It is designed to demonstrate Object-Oriented Programming (OOP) concepts.
- The system allows users to perform basic ATM operations through a menu-driven interface.
- User interaction is handled using console input and output.
- The project is suitable for beginners learning Python classes and methods.

## Features
- Create a new ATM PIN
- Change an existing PIN
- Check account balance
- Withdraw cash from the account
- Exit the system safely
- Menu-driven and user-friendly interface

## Concepts Used
- Classes and Objects
- Constructor (`__init__`)
- Instance variables
- Methods and functions
- Conditional statements
- Loops and user input handling

## Technologies Used
- Python 3

## How to Run
- Install Python 3 on your system
- Clone or download the repository
- Open a terminal or command prompt
- Run the file using:
  ```bash
  python Atm.py
